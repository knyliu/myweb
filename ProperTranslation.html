Proper Translation


A tool specifically designed for the translation of proper nouns in documents.

This project, integrating data from the National Academy for Educational Research and programming, addresses the issue of current translation tools on the market being unable to accurately and correctly translate proper nouns.




# Python Flask
# Python Data Processing
# Team Project

Why Do We Need Proper Translation?



Whether it's Google Translate, ChatGPT, or even other translation tools,

none can translate proper nouns perfectly to meet the standard correct translation of the National Academy for Educational Research.

Reliable Data Sources


This website is built on bilingual data from the National Academy for Educational Research.

It helps you get the most accurate translations of proper nouns in the simplest way.






Trusted Basic Translation Sources


The underlying translation is powered by Microsoft Azure Natural Language Process.

Therefore, you can fully trust the accuracy of the translation.
